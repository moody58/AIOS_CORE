# 1. Scopo del Project Index

Il Project Index è il registro ufficiale dei progetti gestiti dal
sistema AIOS.

Il suo scopo è:\
• mantenere visibilità sull'ecosistema dei progetti\
• evitare dispersione di iniziative\
• permettere interrogazioni rapide dello stato dei progetti\
• collegare la Control Room ai sistemi operativi dei singoli progetti

Il Project Index non contiene i progetti ma li registra e li descrive
sinteticamente.

# 2. Struttura del registro

Ogni progetto registrato nel sistema AIOS include le seguenti
informazioni:

• Nome progetto\
• Stato (Focus / Attivo / Incubazione / Archivio)\
• Nodo operativo principale\
• Posizione nel filesystem\
• Note sintetiche

Questa struttura consente di mantenere il registro leggero e facilmente
consultabile.

# 4. Stati possibili dei progetti

Focus\
Progetti su cui si sta lavorando attivamente.

Attivo\
Progetti esistenti ma non nel focus corrente.

Incubazione\
Idee o iniziative ancora in fase di valutazione (Radar).

Archivio\
Progetti conclusi o non più attivi nel sistema.

# 5. Regole di gestione del Project Index

• ogni nuovo progetto viene registrato nel Project Index\
• lo stato del progetto deve essere coerente con la dashboard AIOS\
• il Project Index non sostituisce i documenti dei progetti\
• il registro viene aggiornato quando nasce o cambia stato un progetto
